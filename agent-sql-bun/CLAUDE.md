# agent-sql (Bun implementation — legacy during Go migration)

This is the original TypeScript/Bun implementation. The Go rewrite is happening at the repo root.
See `../AGENTS.md` for the current project docs.

## Dev tools

- **Linting**: `bun run lint` / `bun run lint:fix`
- **Formatting**: `bun run format`
- **Testing**: `bun test`
- **Typecheck**: `bun run typecheck`
- **Dev runner**: `bun run dev -- <args>`

## Config

Shared with the Go version at `~/.config/agent-sql/config.json`.
